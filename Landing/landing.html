Pelajaran

Pemograman Web adalah Proses pembuatan intruksi-intruksi untuk menciptakan program/aplikasi berbasis internet yang dapat diakses melalui browser.

Materi

Pada materi ini kita akan belajar bagaimana cara membuat web, seiring perkembangan teknologi, web telah berkembang menjadi kompleks.
Dalam membuat web ada 2 bagian tertentu yang harus dibagun yaitu frontend dan backend.

Front End

Frontend adalah bagian dari web yang bisa dilihat oleh user.
User juga bisa berinteraksi pada bagian ini, komponen yang membagun bagian ini adalah :

  1. HTML
  2. CSS
  3. Javascript

Back End

Backend adalah bagian belakang layar dari web.
Disini user tidak bisa berinteraksi karena bagian ini ada di sisi server bukan sisi client. komponen yang membagun bagian ini adalah:

  1. Bahasa Pemograman
  2. Database
  3. Web Server
  4. Web Service
